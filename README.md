# 🧮 Calculator Web Application

A simple, responsive, and user-friendly Calculator Web Application built using **HTML**, **CSS**, and **JavaScript** as part of the **CodSoft Web Development Internship (Task 3)**.

---

## 📌 Features

- ➕ Addition
- ➖ Subtraction
- ✖️ Multiplication
- ➗ Division
- 🧹 Clear Function
- ⌫ Delete Function
- 📱 Responsive Design
- ⚡ Fast & Smooth User Interface

- HTML5
- CSS3
- JavaScript (ES6)
